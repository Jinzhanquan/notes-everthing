## docker 加速

由于国内网络原因，DOCKER访问docker hub经常超时，为了能够给DOCKER加速，我们可使用国内DOCKER镜像；

1. https://www.daocloud.io/ 到该网站注册账户并登陆；
1. https://www.daocloud.io/mirror#accelerator-doc 查看该网页获得设置方法

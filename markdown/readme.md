## 超链接

格式：
```
[文字](url)
```
如：
```
[官网](https://www.baidu.com)
```
效果：
[官网](https://www.baidu.com)

## 图片
格式：
```
[alt](url_or_path)
```
如：
```
[alt](assets/2017-08-21-21-00-25.png)
```
效果：
![alt](assets/2017-08-21-21-00-25.png)
